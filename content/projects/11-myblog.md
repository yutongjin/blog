## Overview

`MyBlog` was one of my early attempts at owning a personal publishing surface. The repo description explicitly calls it a personal blog built with React, and the top-level files show both frontend code and an LNMP deployment bundle, which makes it more operational than a simple UI demo.

## What It Represents

- an early personal-site build
- React as the presentation layer
- hands-on deployment thinking instead of frontend-only work

## Stack

- React
- JavaScript
- HTML/CSS
- LNMP deployment tooling
