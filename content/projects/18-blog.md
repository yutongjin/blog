## Overview

This is the current blog source repo: Hugo content, theme customizations, GitHub Actions deployment, and the private iPhone publishing flow I built later. Compared with the older blog repos, this version is much more intentional about content structure and deployment boundaries.

## What It Represents

- the current writing system
- cleaner separation between source, build, and deploy
- custom tooling for posting from mobile without relying on the GitHub UI

## Stack

- Hugo
- PaperMod
- GitHub Actions
- GitHub Pages
- Python publisher service
- Tailscale Funnel
